Test suite is a collection of test classes. 
It is used to run multiple test classes together. 
It is useful when you want to run all the test classes together. 
You can create a test suite by going to the test tab in the developer console and then click on the new test suite button  and then select the test classes you want to include in the test suite. 
You can also run the test suite by clicking on the run button in the test suite.
